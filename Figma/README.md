# Figma reference materials

В этой папке лежат exports и исходные материалы из Figma: full-page макеты, SVG-блоки, PNG/JPG assets и `.fig` файлы.

Эти файлы используются для сверки верстки с макетом. Они не являются production assets сайта: рабочие изображения и иконки для заданий лежат внутри `task-*/assets/`.
